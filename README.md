# React-Native-Country-Flags-Search
**Mobile version of Country Flags Search.**<br/>A simple website for country searching made with React JS, Tailwind CSS and REST Countries API.

**MAIN FEATURES** <br/>
-API data from REST Countries <br/>
-Data searching <br/>
-Dark mode <br/>
